whoami

Hey, I’m 0ctane — a final-year school student trying to learn ethical hacking,binary reverse engineering and forensics.
Voidstack is my first proper web project — a simple, free platform to help others get started without the chaos I went through.

When I first got into this space, I spent more time finding good resources than actually learning. It felt like every path was either paywalled, too surface-level, or just clickbait. I built Voidstack so no one else has to waste that time. It’s not perfect — and I’m still learning — but it’s real.

what I’ve done so far

  • Built this entire site from scratch using Jekyll, GitHub Pages, TailwindCSS, and deployed a separate Vite version with Netlify
  • Configured my own custom DNS records, reverse proxy setups, and Cloudflare protection
  • Self-hosted media servers like Jellyfin, private web apps, and development environments
  • Set up basic CI/CD pipelines with GitHub Actions for deployments
  • Used Docker for containerized services and dabbled with VMs for isolated testing
  • Comfortable with Java, Bash scripting, and Linux server management
  • Learning my way through Hack The Box, TryHackMe, and CTF platforms — documenting every step here
  • Planning a full Raspberry Pi-based homelab, internal networking, AD setup, and maybe even a personal VPN and logging stack
  • And yeah, when I’m not knee-deep in configs and exploits, I occasionally queue up for CS2 Premier and try to clutch a round or two

why this exists

I’m not here to sell anything or pretend I know everything — because I don’t. I’m still learning every day. This site is part of that process.
If it helps even one beginner avoid the confusion I faced — mission complete.

— 0ctane
